Parte A — HTML
1. ¿Cuál es la función de HTML en un proyecto web?

HTML se encarga de definir la estructura y contenido de una página web mediante etiquetas como encabezados, párrafos, imágenes, formularios y enlaces.

2. ¿Qué error HTML encontraste en el proyecto?

Se encontró un error en la clase hero-contnet, ya que estaba mal escrita y no coincidía con la clase CSS hero-content.

3. ¿Qué ocurre cuando una ruta HTML está incorrecta?

Parte B — CSS
4. ¿Cuál es la función de CSS?

CSS se utiliza para aplicar estilos visuales como colores, tamaños, posiciones, animaciones y diseño responsive.

5. ¿Qué error CSS encontraste?

La propiedad display: grips; estaba incorrecta. La propiedad correcta era display: grid;.

6. ¿Por qué fallaba el diseño responsive?

Porque la media query tenía un error ortográfico: max-wdith en lugar de max-width.

10. ¿Por qué es importante revisar nombres y rutas?

Porque permite detectar errores rápidamente y evitar que varios problemas se acumulen.
